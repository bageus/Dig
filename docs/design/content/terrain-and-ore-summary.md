# Краткая таблица грунта и руд

Основные системные правила находятся в [`../terrain-resource-output-and-processing.md`](../terrain-resource-output-and-processing.md).

## Грунт

| Terrain | Добываемость | Outputs |
|---|---:|---|
| Песчаный | да | нет |
| Каменная порода | да | только камень |
| Металлосодержащая/«серная» | да | камень, железная руда, редкая золотая руда, уголь |
| Кристаллическая | да | мало камня, железная руда, кристаллическая руда, редкая золотая руда |
| Лавовая | да | золотая руда, немного камня и кристаллической руды, железная руда, уголь |
| Недобываемая | нет | нет |

## Цепочки

- железная руда -> железный слиток;
- золотая руда -> золото;
- кристаллическая руда -> кристалл;
- «металл» = железный слиток.

## Здания

- горн: `3 iron ore + 2 mushroom legs -> 2 iron`;
- литейный цех: `3 iron ore + 2 coal -> 2 iron`;
- литейный цех: `3 gold ore + 2 coal -> 2 gold`;
- crystal processor: `1 crystal ore -> 1 crystal`.
